# Excel Sales Dashboard

## Project Overview

This project analyzes sales data using Microsoft Excel to identify sales trends and create an interactive dashboard.

## Skills Demonstrated

- Data Cleaning
- PivotTables
- PivotCharts
- Dashboard Design
- Data Visualization
- Excel Functions (SUM, AVERAGE, MAX, MIN, COUNT)

## Key Insights

- South generated the highest total sales.
- Sarah was the top-performing salesperson.
- Electronics generated the highest revenue.
